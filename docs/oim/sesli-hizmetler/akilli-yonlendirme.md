# Akıllı Yönlendirme Servisi

Numara meşgul veya ulaşılamaz olduğunda aramanın otomatik olarak yönlendirilmesini sağlar.

## Yönlendirme Senaryoları

| Durum | Yönlendirme Hedefi |
|-------|--------------------|
| Meşgul | 0532 111 22 33 (cep telefonu) |
| Cevap yok (30 sn) | Telesekreter |
| Ulaşılamıyor | 0212 555 00 00 (çağrı merkezi) |

## Kural Oluşturma

1. **Akıllı Yönlendirme Servisi** sayfasını açın
2. **Yeni Kural Ekle** butonuna tıklayın
3. Koşulu seçin (Meşgul / Cevap yok / Ulaşılamıyor)
4. Yönlendirme hedefi girin
5. **Kaydet**'e tıklayın

::: tip
Birden fazla kural ekleyebilirsiniz. Kurallar yukarıdan aşağıya sırayla değerlendirilir.
:::
