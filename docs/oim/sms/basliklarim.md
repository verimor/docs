# Başlıklarım

SMS gönderimlerinde gönderici adı olarak görünen alfanümerik başlıklarınızı yönetin.

## Mevcut Başlıklar

| Başlık | Tür | Durum | Onay Tarihi |
|--------|-----|-------|-------------|
| FIRMAADI | Alfanümerik | Aktif | 10.01.2024 |
| BILGI | Alfanümerik | Aktif | 10.01.2024 |
| KAMPANYA | Alfanümerik | Aktif | 15.02.2024 |

## Yeni Başlık Başvurusu

1. **Yeni Başlık Talep Et** butonuna tıklayın
2. İstediğiniz başlık adını girin (maks. 11 karakter, harf ve rakam)
3. Kullanım amacını belirtin
4. Başvuruyu gönderin

Başlıklar **BTK onayından** sonra aktif hale gelir. Onay süresi genellikle 1–3 iş günüdür.

::: warning
BTK kararı gereği API üzerinden kampanya gönderimlerinde yalnızca alfanümerik başlıklar kullanılabilir. Rakamdan oluşan başlıklar bu amaçla kullanılamaz.
:::

## İYS Marka Kodu

Ticari SMS gönderebilmek için başlığınıza İYS marka kodu tanımlanmalıdır. **Düzenle** butonuyla marka kodunu güncelleyebilirsiniz.
