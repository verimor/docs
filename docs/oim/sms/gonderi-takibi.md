## Durum Açıklamaları

| Durum | Açıklama |
|-------|----------|
| İletildi | Alıcı telefonuna ulaştı |
| Bekliyor | Gönderildi, operatör teyidi bekleniyor |
| İletilemedi | Numara kapalı, kara listede veya geçersiz |
| Zaman Aşımı | Geçerlilik süresi içinde iletilemedi |

## Zamanlanmış Gönderimi İptal Etme

Henüz gönderilmemiş (zamanlanmış) kampanyalar listede **İptal Et** butonu ile iptal edilebilir.
