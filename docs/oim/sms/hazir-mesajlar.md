# Hazır Mesajlar

Sık kullandığınız mesaj şablonlarını kaydedin, gönderim sırasında tek tıkla ekleyin.

## Mevcut Şablonlar

| Şablon Adı | Mesaj Önizleme |
|------------|----------------|
| Sipariş Alındı | "Siparişiniz alındı! No: {siparis_no}. Teşekkürler." |
| Kargo Bildirimi | "Siparişiniz kargoya verildi. Takip kodu: {takip_no}" |
| Randevu Hatırlatma | "{tarih} tarihli randevunuzu hatırlatırız." |
| Ödeme Onayı | "Ödemeniz alındı. Tutar: {tutar} TL" |

## Yeni Şablon Ekleme

1. **Yeni Şablon** butonuna tıklayın
2. Şablon adını girin
3. Mesaj metnini yazın — değişken alanlar için `{degisken_adi}` kullanın
4. **Kaydet**'e tıklayın

## Şablon Kullanımı

SMS Gönder sayfasında mesaj alanının yanındaki **Hazır Mesaj Seç** butonuna tıklayın, listeden şablonu seçin. Şablon mesaj alanına yapıştırılır; değişken alanları doldurun.
